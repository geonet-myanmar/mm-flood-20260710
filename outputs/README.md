# Outputs

Products generated by `flood_inundation_s1.py` for the default Central Myanmar AOI.

## Included in the repository

| File | Size (approx.) | Description |
|------|----------------|-------------|
| [`flood_inundation_map.jpg`](flood_inundation_map.jpg) | ~1.2 MB | **Primary publication figure** (JPEG, quality 95, 220 dpi) |
| [`flood_inundation_map.png`](flood_inundation_map.png) | ~2.9 MB | Same figure as PNG |
| [`flood_classification.tif`](flood_classification.tif) | ~1.5 MB | Classified map (GeoTIFF, EPSG:32647, 30 m) |
| [`flood_stats.json`](flood_stats.json) | < 1 KB | Thresholds, pixel counts, areas (km²) |
| [`stac_items.json`](stac_items.json) | < 1 KB | STAC item IDs, AOI, dates |

## Not version-controlled

| File | Reason |
|------|--------|
| `vv_db.npz` | Large intermediate (~100+ MB); regenerated automatically |

## GeoTIFF class values

| Value | Class | Suggested colour |
|------:|-------|------------------|
| 0 | No data / land / other | Transparent or white |
| 1 | Flood inundation (new water) | Cyan |
| 2 | Permanent / stable water | Blue |

CRS: **EPSG:32647** (WGS 84 / UTM zone 47N). Pixel size: **30 m**.
